# esp32-prototype-board

EAGLE schematic for prototype board with 12V power input and all essentials.

Programmed and debugged using [ESP-PROG](https://github.com/espressif/esp-dev-kits/blob/master/esp-prog/docs/ESP-Prog_guide_en.md) interface.
